üöÄ Quick Start

Installation

Get started with DeshChain in minutes

# Install deshchaind
curl -L https://get.deshchain.com | bash

# Initialize node
deshchaind init mynode --chain-id deshchain-1

# Start node
deshchaind start
Full Installation Guide ‚Üí

Create Your First Transaction

Send NAMO tokens on DeshChain

# Create wallet
deshchaind keys add mywallet

# Check balance
deshchaind query bank balances [address]

# Send transaction
deshchaind tx bank send [from] [to] 100namo
Transaction Guide ‚Üí

Deploy Smart Contract

Deploy your first smart contract

# Compile contract
cargo build --target wasm32-unknown-unknown

# Deploy to chain
deshchaind tx wasm store contract.wasm

# Instantiate
deshchaind tx wasm instantiate [code-id]
